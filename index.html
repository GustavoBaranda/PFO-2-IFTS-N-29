Sobre Mí

Hola, soy Gustavo, un apasionado desarrollador FrontEnd con más de 5 años de experiencia en la creación de interfaces de usuario modernas, atractivas y funcionales. A lo largo de mi carrera he trabajado con una variedad de tecnologías, desde HTML y CSS hasta JavaScript y frameworks como React. Mi enfoque siempre está en la experiencia del usuario (UX), asegurando que cada proyecto sea intuitivo y fácil de usar.

Mi pasión por la programación comenzó desde joven, y con el tiempo me he especializado en crear soluciones innovadoras que mejoren la interacción digital. Me encanta aprender nuevas herramientas y tecnologías, y disfruto enfrentando desafíos técnicos para crear aplicaciones web de alto rendimiento.

Además de mi experiencia técnica, tengo un gran interés por el diseño visual, lo que me permite combinar desarrollo y estética para crear productos digitales completos. Trabajo de cerca con diseñadores y otros desarrolladores para asegurar que todos los aspectos del proyecto estén alineados y contribuyan a una experiencia de usuario excepcional.

Actualmente, me encuentro buscando nuevos proyectos y oportunidades de colaboración que me permitan seguir creciendo como profesional y aportar mis habilidades a equipos innovadores. Si estás interesado en trabajar juntos o quieres saber más sobre mis proyectos anteriores, ¡no dudes en contactarme!

Proyectos

Las Verde

"Las Verde" es una aplicación web informativa sobre la fabricación de jugos orgánicos. Presenta la historia, valores, procesos productivos y catálogo de productos del emprendimiento, con conexión a Firebase para la gestión de datos. Utiliza ReactJS, Sass para los estilos y React Router Dom para la navegación.

Descubriendo Jardin Maternal

Este sitio web es una plataforma informativa sobre un jardín maternal, donde se presentan su misión, valores, metodología de enseñanza y formas de contacto. Desarrollado con HTML, CSS y JavaScript Vanilla, incorpora PHP para la gestión de formularios de contacto.

Chetifabene

CHETIFABENE es una red social diseñada para compartir, guardar y publicar recetas de cocina, permitiendo seguir a otros usuarios y puntuar sus comidas favoritas. Desarrollada con React, Node.js y PostgreSQL, utiliza Cloudinary para almacenamiento de imágenes y Nodemailer para notificaciones. La aplicación ofrece una interfaz intuitiva y está desplegada en Vercel. Fue creada como solución laboral en No Country.
